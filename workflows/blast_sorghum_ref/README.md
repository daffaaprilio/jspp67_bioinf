### BLAST genes of interest against sorghum proteome

To obtain the homologs of these genes (often referred to as bait genes), then use them to query the sorghum coexpression database.

You may wonder why this workflow module has a more recent timestamp than its successor, `hmm_homology`. The explanation: the newer module is indeed preferred after performing this, but somehow I lost this module, so the current file (this `blast_sorghum_ref/README.md` as well as the `blast_sorghum_ref/Snakefile`) are made to compensate that.
